Productos
Server migration
Migraciones del proceso computacional local hacia la nube. Permite un fácil acceso a la información, seguridad de los datos y costos adecuados en base al uso de los recursos computacionales.
EC2
Capacidad de cómputo virtual y adaptable a las características del las aplicaciones o soluciones del cliente.
Amazon RDS
Servicio web que se ejecuta en la nube diseñado para simplificar la configuración, el funcionamiento y el escalonamiento de bases de datos relacionales para el uso de aplicaciones.
Amazon S3
También conocido como simple storage servicie es un servicio que proporciona almacenamiento de objetos a través de una interfaz de servicio web.
